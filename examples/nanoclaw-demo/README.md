# examples/nanoclaw-demo

Minimal NanoClaw + ARP integration — shows both `arpGuardedTool()` (for wrapping plain async functions in constrained environments) and `withArp(nano, opts)` (for NanoClaw-like hosts that expose a registerable hook).

See `../../adapters/nanoclaw/MIGRATION.md` and the full conformance coverage in `tests/phase-6/adapter-conformance.test.ts`.
