# Getting started

Two paths depending on what you're trying to do:

## I want to use ARP

Go to [**Quick start**](/docs/quickstart) — install ARP and pair two agents in about 10 minutes.

For a complete two-person walkthrough with all the wire-level detail, see [**Example setup**](/docs/example-setup).

## I want to build with ARP

Go to [**Architecture**](/spec/v0.1/architecture) — the system design overview, then the [Spec](/spec/v0.1/overview) for normative protocol details, then [SDKs](/docs/sdks) and [Framework adapters](/docs/adapters) for integration code.

## I want to deploy ARP myself

Go to [**Install overview**](/docs/install) — three deployment modes (managed cloud, sidecar, library) with tradeoffs for each.
